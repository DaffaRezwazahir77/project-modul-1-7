Tugas Web Developer

Front-End Developer

Mengoptimalkan Fungsi Aplikasi berdasarkan User Interface (UI) dan User Experience (UX).

Back-End Developer

Berjalan di sisi server dengan mengelola data menjadi informasi dari database.

Full-Stack Developer

Memiliki kemampuan di kedua sisi, yaitu front-end dan back-end, sehingga dapat menangani semua aspek pengembangan web.
